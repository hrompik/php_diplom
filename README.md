# php_diplom
Работа на
* Nginx_1.23
* Apache_2.4 (.htaccess лежит в public)
* PHP_8.1
* MySQL-8.0-Win10
почта debugmail.io


Установка
* git clone
* composer i
* изменение файла .env
* php bin/console doctrine:database:create
* php bin/console doctrine:migrations:migrate
* php bin/console doctrine:fixtures:load
